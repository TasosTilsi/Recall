Visit http://localhost:8765

Confirm:
- [ ] Page loads (no blank screen, no 404)

**Step 4 — Verify scope toggle:**
- [ ] Click "Global" button → graph reloads (spinner or brief empty state)
- [ ] Click "Project" button → graph reloads back to project scope
- [ ] Terminal stays running — no restart required

**Step 5 — Verify search:**
- [ ] Type a name fragment in the search input → non-matching nodes dim (become darker/less visible)
- [ ] Clear search → all nodes return to normal color

**Step 6 — Verify read-only (no edit controls visible):**
- [ ] No "Edit", "Delete", "Add", "Save" buttons anywhere in the UI
- [ ] Right-clicking a node shows no context menu with write actions

**Step 7 — Verify global scope (UI-03):**
Stop the current server (Ctrl+C), then:
```bash
graphiti ui --global
```
Expected banner shows "Scope: global". Visit http://localhost:8765. Confirm different (or same if graphs overlap) set of nodes from global scope.

**Step 8 — Verify Ctrl+C shutdown:**
Press Ctrl+C in the terminal.
Expected: "Graphiti UI stopped." message, terminal returns to prompt cleanly.

**Step 9 — Verify port conflict error:**
Start the server, then in another terminal:
```bash
graphiti ui --api-port 8765
```
Expected: Error message "Port 8765 is already in use..." and exit.
