```python
"""indexer.py — Incremental git history indexing control flow.

Entry points:
    run_init(repo_root)  — full rebuild, clears and repopulates DB
    run_sync(repo_root)  — incremental: only commits newer than last_indexed_sha
"""
from __future__ import annotations

import sqlite3
import uuid
from pathlib import Path
from typing import Optional

import structlog

from src.config import Config, load_config
from src.db.manager import DatabaseManager
from src.extractor.engine import extract_batch
from src.extractor.git_walker import CommitRecord, walk_commits, batch_commits

logger = structlog.get_logger()

_LAST_SHA_KEY = "last_indexed_sha"
```

Implement `_get_db(repo_root: Path, config: Config) -> DatabaseManager`:
- Returns `DatabaseManager(config)`.
- Does NOT call init_db() — callers decide when to initialise.

Implement `_read_last_sha(conn: sqlite3.Connection) -> str | None`:
- Executes `SELECT value FROM metadata WHERE key = ?` with `(_LAST_SHA_KEY,)`.
- Returns `row["value"]` if row exists, else `None`.

Implement `_write_last_sha(conn: sqlite3.Connection, sha: str) -> None`:
- Executes `INSERT OR REPLACE INTO metadata (key, value) VALUES (?, ?)` with `(_LAST_SHA_KEY, sha)`.
- Calls `conn.commit()`.

Implement `_insert_entities(conn: sqlite3.Connection, entities: list[dict]) -> int`:
- For each entity dict with keys type, name, content, commit_sha:
  - Generate id: `str(uuid.uuid5(uuid.NAMESPACE_DNS, f"{entity['type']}:{entity['name']}"))`
  - Execute `INSERT OR IGNORE INTO entities (id, type, name, content, commit_sha) VALUES (?, ?, ?, ?, ?)`
- Calls `conn.commit()` once after all insertions.
- Returns count of rows actually inserted (use `conn.execute(...).rowcount` per insert, sum them).

Implement `_commits_after_sha(commits: list[CommitRecord], sha: str) -> list[CommitRecord]`:
- `commits` is oldest-first from walk_commits.
- Find the index of the commit with `commit.sha == sha`.
- Return all commits AFTER that index (i.e., `commits[idx+1:]`).
- If sha not found in list (e.g., SHA is from before current git history), return all commits (safe fallback — log a warning).
- If sha found at end of list (nothing newer), return `[]`.

Implement `run_init(repo_root: Path, config: Optional[Config] = None) -> dict`:
1. `config = config or load_config()`
2. `db = _get_db(repo_root, config)`
3. `db.init_db()`  — idempotent, safe to call always
4. `commits = walk_commits(repo_root)`
5. `batch_size = getattr(getattr(config, 'indexer', None), 'batch_size', 10)`
6. `batches = batch_commits(commits, batch_size)`
7. `total = len(commits)`; `entities_inserted = 0`; `commits_processed = 0`
8. With `db.connect() as conn`:
   - DELETE last_indexed_sha: `conn.execute("DELETE FROM metadata WHERE key = ?", (_LAST_SHA_KEY,)); conn.commit()`
   - For i, batch in enumerate(batches):
     - `start = i * batch_size + 1`; `end = min(start + len(batch) - 1, total)`
     - `logger.info("indexing_batch", range=f"{start}–{end}", total=total)`
     - `entities = extract_batch(batch)`
     - `entities_inserted += _insert_entities(conn, entities)`
     - `commits_processed += len(batch)`
   - If commits: `_write_last_sha(conn, commits[-1].sha)`  — HEAD is last in oldest-first list
9. `logger.info("init_complete", commits_processed=commits_processed, entities_inserted=entities_inserted)`
10. Return `{"commits_processed": commits_processed, "entities_inserted": entities_inserted}`

Implement `run_sync(repo_root: Path, config: Optional[Config] = None) -> dict`:
1. `config = config or load_config()`
2. `db = _get_db(repo_root, config)`
3. If `not db.get_db_path().exists()`:
   - `logger.info("no_db_found_running_init")`
   - Return `run_init(repo_root, config)`
4. `db.init_db()`  — ensure schema is up to date
5. With `db.connect() as conn`: `last_sha = _read_last_sha(conn)`
6. `all_commits = walk_commits(repo_root)`
7. If `last_sha`:
   - `new_commits = _commits_after_sha(all_commits, last_sha)`
8. Else:
   - `new_commits = all_commits`  (no cursor yet — process everything)
9. If `len(new_commits) == 0`:
   - `print("0 commits to process")`
   - `logger.info("sync_complete", commits_processed=0, entities_inserted=0)`
   - Return `{"commits_processed": 0, "entities_inserted": 0}`
10. `batch_size = getattr(getattr(config, 'indexer', None), 'batch_size', 10)`
11. `batches = batch_commits(new_commits, batch_size)`
12. `total = len(new_commits)`; `entities_inserted = 0`; `commits_processed = 0`
13. With `db.connect() as conn`:
    - For i, batch in enumerate(batches):
      - `start = i * batch_size + 1`; `end = min(start + len(batch) - 1, total)`
      - `logger.info("syncing_batch", range=f"{start}–{end}", total=total)`
      - `entities = extract_batch(batch)`
      - `entities_inserted += _insert_entities(conn, entities)`
      - `commits_processed += len(batch)`
    - `_write_last_sha(conn, new_commits[-1].sha)`  — most recent processed
14. `logger.info("sync_complete", commits_processed=commits_processed, entities_inserted=entities_inserted)`
15. Return `{"commits_processed": commits_processed, "entities_inserted": entities_inserted}`
