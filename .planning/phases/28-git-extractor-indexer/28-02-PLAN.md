  <name>Task 1: Extraction engine — extract_batch with LLM call, JSON parse, normalization</name>
  <files>src/extractor/engine.py, tests/extractor/test_engine.py</files>
  <behavior>
    - extract_batch(batch: list[CommitRecord]) -> list[EntityRecord]
    - Calls build_batch_prompt(batch) to produce prompt string
    - Invokes LLM via subprocess: runs `claude -p "{prompt}"` using shutil.which("claude"); captures stdout
    - Parses stdout as JSON; expects {"entities": [...]} structure matching ExtractionResult
    - Filters out any entity records where type is not in VALID_ENTITY_TYPES
    - Normalizes each entity name: entity["name"].lower().strip()
    - Sets commit_sha on each entity to the first commit in the batch sha (the batch-level sha attribution)
    - On json.JSONDecodeError: logs warning via structlog with batch size and raw response preview (first 200 chars), returns []
    - On subprocess CalledProcessError or TimeoutExpired (60s timeout): logs error, returns []
    - On empty batch: raises ValueError("batch must not be empty")
    - Test: with mock subprocess returning valid JSON, returns normalized entities
    - Test: with mock subprocess returning malformed JSON, returns [] without raising
    - Test: entities with invalid type are dropped from result
    - Test: entity names are lowercased and stripped
  </behavior>
  <action>
    Create `src/extractor/engine.py`:

    ```python
    import json
    import shutil
    import subprocess
    from typing import List
    import structlog
    from src.extractor.git_walker import CommitRecord
    from src.extractor.prompt import build_batch_prompt, VALID_ENTITY_TYPES, EntityRecord

    logger = structlog.get_logger()

    LLM_TIMEOUT_SECONDS = 60
    ```

    Implement `extract_batch(batch: list[CommitRecord]) -> list[EntityRecord]`:
    1. Raise ValueError if batch is empty
    2. prompt = build_batch_prompt(batch)
    3. Locate claude binary: `claude_bin = shutil.which("claude")`; if None raise RuntimeError("LLM provider 'claude' not available — install claude CLI")
    4. Run: `result = subprocess.run([claude_bin, "-p", prompt], capture_output=True, text=True, timeout=LLM_TIMEOUT_SECONDS, check=True)`
    5. Parse `result.stdout` as JSON. On JSONDecodeError: log warning with `batch_size=len(batch)` and `response_preview=result.stdout[:200]`, return [].
    6. Extract `data.get("entities", [])`. Filter to only items where `item.get("type") in VALID_ENTITY_TYPES`. For each kept item, normalize: `item["name"] = item.get("name", "").lower().strip()`. Set `item["commit_sha"] = batch[0].sha` if not already set.
    7. Log info: `entities_extracted=len(result_list), batch_commits=[c.short_sha for c in batch]`
    8. Return normalized list.

    Create `tests/extractor/test_engine.py` using `unittest.mock.patch("subprocess.run")` to simulate LLM responses. Test all cases from behavior block.
