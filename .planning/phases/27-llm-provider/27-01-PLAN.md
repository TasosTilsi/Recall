    Create `src/config.py` as the single v3.0 config module. Do NOT modify src/llm/config.py — Phase 25 teardown removes it; this file is the replacement.

    Define three frozen dataclasses:

    ```python
    @dataclass(frozen=True)
    class LLMConfig:
        provider: str           # "claude" | "ollama" | "openai"
        model: str              # e.g. "claude-haiku-4-5-20251001"
        url: str                # base URL; "" means use provider default
        api_key: str            # Bearer token; "" means none

    @dataclass(frozen=True)
    class EmbeddingsConfig:
        provider: str           # "ollama" | "openai"
        model: str
        url: str
        api_key: str

    @dataclass(frozen=True)
    class DBConfig:
        path: str               # relative or absolute path string

    @dataclass(frozen=True)
    class Config:
        llm: LLMConfig
        embeddings: EmbeddingsConfig | None   # None = embeddings disabled
        db: DBConfig
    ```

    Define `load_config(config_path: Path | None = None) -> Config`:
    - Default config_path: `Path.home() / ".recall" / "config.toml"`
    - If file does not exist: return Config with defaults (no error)
    - Read [llm] section: provider (default "claude"), model (default "claude-haiku-4-5-20251001"), url (default ""), api_key (default "")
    - Validate provider in {"claude", "ollama", "openai"}: raise ValueError(f"Invalid LLM provider '{val}'. Valid: claude, ollama, openai") if not
    - For [llm] with provider="ollama" and url="" → set url="http://localhost:11434"
    - For [llm] with provider="openai" and url="" → leave url="" (user must configure)
    - Read [embeddings] section: if absent → Config.embeddings = None. If present → EmbeddingsConfig with provider, model, url (default "http://localhost:11434"), api_key (default "")
    - Read [db] section: path (default ".recall/recall.db")
    - Use `tomllib` (stdlib Python 3.11+) — NOT tomli or toml packages
    - Do NOT import from src.llm.* — this module is standalone

    Use structlog.get_logger(__name__) for any warnings (e.g. unknown keys). No logging in tests.

    Write `tests/test_config_v3.py` covering all 6 behaviors above. Use tmp_path fixture to write temp toml files. Use pytest (no asyncio needed — sync module).
