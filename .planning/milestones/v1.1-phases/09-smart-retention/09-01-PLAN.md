  <name>Task 1: Create RetentionManager module (src/retention/)</name>
  <files>src/retention/__init__.py, src/retention/manager.py</files>
  <behavior>
    - get_retention_manager() returns same instance on repeated calls (singleton)
    - reset_retention_manager() clears singleton (used in tests)
    - RetentionManager.__init__ creates ~/.graphiti/ if missing, creates DB, runs schema
    - DB has three tables: access_log (uuid, scope, last_accessed_at, access_count PRIMARY KEY (uuid,scope)), pin_state (uuid, scope, pinned_at PRIMARY KEY (uuid,scope)), archive_state (uuid, scope, archived_at PRIMARY KEY (uuid,scope))
    - PRAGMA journal_mode=WAL set on every connection via _get_conn()
    - record_access(uuid, scope): upserts access_log — increments access_count, updates last_accessed_at to UTC now
    - pin_node(uuid, scope): inserts into pin_state (ignore if already pinned)
    - unpin_node(uuid, scope): deletes from pin_state
    - archive_node(uuid, scope): inserts into archive_state (ignore if already archived)
    - clear_archive(uuid, scope): deletes from archive_state — used when a previously archived node is reactivated by a new episode; no-op if uuid not in archive_state
    - is_pinned(uuid, scope) -> bool
    - is_archived(uuid, scope) -> bool
    - get_access_record(uuid, scope) -> dict with {last_accessed_at: datetime|None, access_count: int}
    - get_archive_state_uuids(scope) -> set[str] of all archived UUIDs in that scope
    - get_pin_state_uuids(scope) -> set[str] of all pinned UUIDs in that scope
    - compute_score(created_at, last_accessed_at, access_count, retention_days) -> float 0.0-1.0; lower = more stale
    - Score algorithm: age_factor = min(age_days / retention_days, 1.0); recency_factor = max(0.0, 1.0 - days_since_access / retention_days); access_bonus = min(access_count * 0.3 * recency_factor, 0.5); score = age_factor * (1.0 - access_bonus); returns round(score, 3)
    - Timezone handling: always normalize naive datetimes with .replace(tzinfo=timezone.utc) before subtraction
  </behavior>
