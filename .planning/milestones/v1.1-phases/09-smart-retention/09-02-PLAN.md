  <behavior>

      * Get archived_uuids = retention.get_archive_state_uuids(scope_key)
      * If archived_uuids is empty, skip the check (fast path)
      * Otherwise: query EntityNode.get_by_group_ids([group_id], graphiti._driver) to get current entity UUIDs
      * For any entity UUID that appears in both the current entity set AND archived_uuids:
        - Call retention.clear_archive(uuid, scope_key)
        - Log "retention_node_reactivated" with uuid and scope_key via structlog
    - Wrap the entire reactivation block in try/except — reactivation failure must NOT fail the add() call
    - The check is post-episode: it runs after add_episode() completes, before add() returns
    - Performance note: get_archive_state_uuids() returns a set; EntityNode.get_by_group_ids() is already called by list_entities — same cost. Only runs when archived_uuids is non-empty (most users: fast no-op)
  </behavior>
  <action>
    Edit src/graph/service.py — locate the add() method and add the reactivation block after the add_episode() call and before the return statement:

    ```python
    # Reactivation: if any archived entity was matched by this episode, remove it from archive_state
    try:
        from src.retention import get_retention_manager
        scope_key = self._get_group_id(scope, project_root)
        retention = get_retention_manager()
        archived_uuids = retention.get_archive_state_uuids(scope_key)
        if archived_uuids:
            graphiti_instance = await self._get_graphiti(scope, project_root)
            group_id = self._get_group_id(scope, project_root)
            current_entities = await EntityNode.get_by_group_ids([group_id], graphiti_instance._driver)
            for entity in current_entities:
                if entity.uuid in archived_uuids:
                    retention.clear_archive(uuid=entity.uuid, scope=scope_key)
                    logger.info("retention_node_reactivated", uuid=entity.uuid, scope=scope_key)
    except Exception:
        logger.warning("retention_reactivation_check_failed", method="add")
    ```
